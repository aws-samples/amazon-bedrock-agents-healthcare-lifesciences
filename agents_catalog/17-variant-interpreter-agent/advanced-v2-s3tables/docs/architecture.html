🧬 Genomics Variant Interpretation Platform

VCF → VEP Annotation → S3 Tables (Iceberg) → Dynamic SQL via Athena → AI Clinical Interpretation

Key Differentiators
🗄️ S3 Tables (Apache Iceberg)
Open-standard Apache Iceberg tables with automatic compaction, partition pruning by sample + chromosome, and standard SQL via Athena. Scales to billions of variants across thousands of patients. Compatible with Spark, Trino, and pandas.
⚡ Dynamic SQL Generation
Agent retrieves live schema from S3 Tables catalog, then generates SQL dynamically for any question. No hardcoded queries — adapts automatically to schema changes. Input validation blocks DDL/DML, enforces LIMIT.
🔗 A2A Agent Protocol
Deployed on AgentCore Runtime with A2A protocol. Discoverable via Agent Registry. Other agents can invoke variant interpretation as a composable genomics service.
